# React Campo Minado

Jogo do campo minado em React com cronometro.

![Screenshot](contadorTruco.gif)

## Tecnologia
- ReactJs
- Styled-components

## Intalação

```sh
$ git clone https://github.com/ldonizete/react-contador-truco.git
$ cd react-contador-truco
$ npm install
$ npm start
```
